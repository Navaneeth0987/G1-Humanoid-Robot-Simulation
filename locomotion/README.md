# Locomotion Module

## 📌 Description
This module implements walking control for the G1 humanoid robot.

## 📂 File
- `g1_walk.py` → Script for executing walking behavior

## ⚙️ Functionality
- Generates walking motion  
- Controls joint movement for locomotion  
- Maintains balance and stability  

## 🧠 Key Concepts
- Gait generation  
- Balance control  
- Motion coordination  

## 🚀 Usage
Run the script to simulate walking:

```bash
python g1_walk.py
